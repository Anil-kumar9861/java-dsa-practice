# Java Arrays Example

## Project Overview
A simple Java console program demonstrating basic array access.

## Structure
- `Arrays/Main.java` — Single Java class that creates an integer array and prints the element at index 2

## Running the Project
The workflow compiles and runs the Java program:
```
cd Arrays && javac Main.java && java Main
```
Output: `4` (the value at index 2 of `{1,2,4,5,8,7}`)

## Environment
- Language: Java (GraalVM 22.3)
- Nix channel: stable-25_05
- Workflow type: console (no web server)
